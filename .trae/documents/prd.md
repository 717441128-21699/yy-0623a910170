## 1. 产品概述

面向线下剧本杀门店前台和店长的平板端偏好匹配台，用于玩家到店临时拼车时快速判断本格/变格接受度。系统通过问答录入玩家偏好，智能匹配车位组合，生成专业推荐说明，帮助店员以更专业但不冒犯的方式沟通，提升临场成交效率。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用场景 | 核心权限 |
|------|----------|----------|
| 前台/店员 | 日常接待散客拼车 | 录入玩家偏好、组车匹配、查看推荐 |
| 店长 | 管理剧本库与门店配置 | 管理剧本库、查看匹配数据 |

### 2.2 功能模块

1. **问答页**：当日剧本选择、玩家偏好录入、偏好标签生成
2. **组车页**：散客拖拽组车、实时冲突检测、换本建议提示
3. **推荐页**：推荐理由说明、踩雷点警示、玩家沟通话术生成

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 问答页 | 剧本选择区 | 展示门店所有剧本，前台勾选当天可开本子，显示剧本类型标签（本格/变格/社会派等）、人数范围、时长 |
| 问答页 | 玩家录入区 | 逐位录入玩家人数、剧本经验等级（新手/进阶/老手）、雷点（多选标签如"恐怖""情感沉浸""硬核推理"等）、偏好维度滑块 |
| 问答页 | 偏好标签生成 | 自动根据录入数据为每位玩家生成标签：如"硬核推理优先""变格可尝试""强变格谨慎""情感沉浸友好"等 |
| 问答页 | 玩家列表 | 展示已录入玩家卡片，显示姓名、偏好标签、可快速编辑或删除 |
| 组车页 | 散客待选池 | 展示所有已录入但未组车的玩家，以卡片形式呈现，支持拖拽 |
| 组车页 | 车位区域 | 显示可组车的车位（根据当天可选剧本生成），每个车位显示剧本名、已占位/总位数，玩家可拖入 |
| 组车页 | 冲突检测提示 | 实时检测：推理强度差异大、有玩家拒绝灵异设定、情感沉浸偏好冲突、经验差距过大等，以红色/黄色警告显示 |
| 组车页 | 换本建议 | 当冲突严重时，建议"适合换成社会派本格""推荐换变格本"等 |
| 推荐页 | 推荐理由说明 | 自动生成一段简短说明：为什么推荐这车（如"3位玩家都偏好硬核推理，与《雾中列车》本格推理风格高度契合"） |
| 推荐页 | 踩雷点警示 | 列出可能的踩雷点（如"玩家A不接受灵异设定，本剧本含微灵异元素，请提前沟通"） |
| 推荐页 | 沟通话术 | 生成一段店员可直接对玩家说的推荐话术，专业但不冒犯 |

## 3. 核心流程

前台到店后，先在问答页选择当天可开剧本，然后逐位录入到店散客的偏好信息。系统自动为每位玩家生成偏好标签。当有足够散客时，店员进入组车页，将散客拖入对应剧本车位。页面实时检测玩家间偏好冲突并给出提示，严重时建议换本。组车确认后进入推荐页，查看推荐理由和踩雷点，获取沟通话术，方便店员专业地与玩家沟通。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色背景（#0f0f1a 暗夜蓝黑），搭配琥珀金（#d4a44c）作为主要强调色，暗红（#8b2c2c）作为警告/危险色
- **次色调**：暗灰蓝（#1a1f2e）用于卡片背景，浅灰（#9ca3af）用于次要文字
- **按钮风格**：圆角微3D效果，琥珀金按钮用于主要操作，暗色按钮用于次要操作
- **字体**：标题使用"ZCOOL QingKe HuangYou"（站酷庆科黄油体）体现悬疑氛围，正文使用"Noto Sans SC"保证中文可读性
- **布局风格**：平板横屏优先，左侧为操作区、右侧为结果/预览区，卡片式布局
- **图标风格**：使用 Lucide 图标库，线条风格与暗色主题统一
- **动效**：拖拽时卡片浮起阴影、标签出现时轻微弹跳、警告脉冲闪烁

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 问答页 | 剧本选择区 | 横向滚动卡片列表，每张卡片显示剧本封面、名称、类型标签、人数范围；选中时金色边框高亮 |
| 问答页 | 玩家录入区 | 表单式布局，姓名输入、经验等级单选、偏好维度滑块（推理/情感/灵异/机制）、雷点多选标签组 |
| 问答页 | 玩家列表 | 纵向卡片流，每张卡片显示玩家姓名、偏好标签（彩色小标签）、编辑/删除按钮 |
| 组车页 | 散客待选池 | 左侧面板，玩家小卡片纵向排列，支持拖拽 |
| 组车页 | 车位区域 | 右侧主区域，每个车位为横向卡片，内含剧本信息+玩家槽位（虚线框） |
| 组车页 | 冲突提示 | 车位下方浮动条，红色/黄色背景，显示冲突文案+换本建议 |
| 推荐页 | 推荐理由 | 顶部大卡片，展示推荐理由文案，金色强调关键词 |
| 推荐页 | 踩雷点 | 中部警示卡片列表，红色左边框，列出每个踩雷点 |
| 推荐页 | 沟通话术 | 底部可复制文本框，带复制按钮，暗色背景金色边框 |

### 4.3 响应式

- 平板横屏优先（1024×768 起步），适配 iPad / Android 平板
- 触摸优化：拖拽手势、大号点击区域（最小 44px）、滑块粗把手
- 竖屏时左右布局切换为上下布局
